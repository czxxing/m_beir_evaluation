## 快速导航

### 核心概念
- **Queries（查询）**：用户搜索问题，作为检索输入
- **Corpus（语料库）**：待检索的文档集合  
- **Qrels（相关性判断）**：查询与文档的相关性标注，作为评估标准

### 评估指标
- **NDCG@k**：归一化折损累积增益，衡量前k个结果的质量
- **Recall@k**：召回率，衡量找到相关文档的能力
- **MAP**：平均精度均值，综合评估检索性能
- **MRR**：平均倒数排名，关注第一个相关文档的排名
